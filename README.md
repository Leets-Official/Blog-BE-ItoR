# Blog 만들기
- 헥사곤 아키텍쳐에 대해 연습하고 있어서, 과제 또한 헥사곤 아키텍쳐로 구성 해보았습니다!
- 공통 로직(시큐리티, 응답객체, 예외핸들링)은 common으로 정했습니다.
- 도메인 관련 내용은 workspace로 정했습니다.
